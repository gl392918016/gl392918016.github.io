使用hexo构建博客

1、安装node.js和git。


2、新建一个文件blog（名字自取），在文件中右键打开git bash。

3、输入npm install -g hexo-cli,安装hexo。

$ npm install -g hexo-cli

4、输入hexo init，初始化该文件夹。

$ hexo init

5、输入hexo serve，开启本地服务器，（如果端口被占用，输入hexo serve -p 端口号，改变端口）。

$ hexo serve
$ hexo serve -p 5000

6、改变主题，登录hexo官网，点击主题，选择你喜欢的一款主题，点击进去后复制项目地址，输入git clone 项目地址 themes/项目名称，克隆下来主题，并在文件根目录的_config.yml文件里边找到theme:位置改变为项目名称。

$ git clone https://github.com/miccall/hexo-theme-Mic_Theme.git themes/mic

7、根据主题的内容，进行更改配置和创建文章。

8、改变代码高亮hexo-prism-plugin插件，安装命令npm i -s hexo-prism-plugin。

$ npm i -s hexo-prism-plugin

9、然后修改根目录下的_config.yml文件中的highlight.enable的值为false，并新增prism插件相关的配置

prism_plugin:
  mode: 'preprocess'    # realtime/preprocess
  theme: 'tomorrow'
  line_number: false    # default false
